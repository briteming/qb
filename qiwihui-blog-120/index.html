03. 字典 — Python 进阶

Aug 13, 2022#技术6 min. read

字典是无序，可变和可索引的集合。 字典由键值对的集合组成。 每个键值对将键映射到其关联值。 字典用大括号书写。 每对键值均以冒号（ : ）分隔，并且各项之间以逗号分隔。
